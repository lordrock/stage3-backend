# Intelligence Query Engine

## Overview
This project is a backend API built for the Stage 2 Backend Task: Intelligence Query Engine.

It provides a queryable demographic intelligence service that supports:

- advanced filtering
- sorting
- pagination
- natural language query parsing
- database seeding with profile data

The system is designed to help clients search and segment profile datasets efficiently.

## Features
- MongoDB-backed profile storage
- Idempotent database seeding
- Advanced filtering on profiles
- Sorting by supported fields
- Pagination with page and limit
- Rule-based natural language search
- Clean and consistent JSON responses
- CORS enabled for external access

## Tech Stack
- Node.js
- Express.js
- MongoDB Atlas
- Mongoose
- CORS
- UUID
- Nodemon
